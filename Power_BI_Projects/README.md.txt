# Power BI Projects

Welcome to the Power BI Projects folder. This collection features various dashboards showcasing sales data, analysis, and forecasting. Each dashboard is interactive, with KPIs and dynamic visualizations.

## Projects Overview

### 1. **Global Chocolate Factory Sales Dashboard**
- **Project Description**: This dashboard visualizes the sales performance of a global chocolate factory, with insights into revenue, sales volume, and regional performance across different continents.
- **Key Features**:
  - Sales by region and product category (e.g., Dark Chocolate, Milk Chocolate)
  - Revenue and units sold KPIs
  - Interactive filters for different time periods (monthly, quarterly)
- **Files**:
  - `Global_Chocolate_Factory_Sales_Dashboard.pbix` – The Power BI dashboard file
  - `README.md` – Project description
  - `/assets/` – Screenshots and GIFs of the dashboard
    - `dashboard-overview.png` – Full dashboard overview
    - `kpi-interaction.gif` – GIF showing KPI interactivity

### 2. **U.S. Fictitious Beverage Brand Interactive Sales Dashboard with KPI**
- **Project Description**: This project features a sales analysis dashboard for a fictitious U.S. beverage brand, displaying key performance indicators like revenue, average price per unit, and units sold across various states and product categories.
- **Key Features**:
  - Interactive visualization for revenue by month, product, and region
  - Drill-down capabilities for more granular insights
  - KPIs like revenue, average price per unit, and units sold
- **Files**:
  - `US_Fictitious_Beverage_Brand_Sales_Dashboard.pbix` – The Power BI dashboard file
  - `README.md` – Project description
  - `/assets/` – Screenshots and GIFs of the dashboard
    - `dashboard-overview.png` – Full dashboard overview
    - `kpi-interaction.gif` – GIF showing KPI interactivity

### 3. **Chocolate Factory Data Analysis Dashboard**
- **Project Description**: This dashboard provides in-depth analysis of chocolate factory data, including sales, production levels, and operational efficiency across different factory locations.
- **Key Features**:
  - Production performance vs. sales analysis
  - Inventory and operational efficiency KPIs
  - Time series analysis to identify seasonal trends
- **Files**:
  - `Chocolate_Factory_Data_Analysis_Dashboard.pbix` – The Power BI dashboard file
  - `README.md` – Project description
  - `/assets/` – Screenshots and GIFs of the dashboard
    - `dashboard-overview.png` – Full dashboard overview
    - `kpi-interaction.gif` – GIF showing KPI interactivity

### 4. **2025 Estimated Population by Country Dashboard**
- **Project Description**: This Power BI dashboard presents an estimation of the global population for 2025 by country. It allows users to explore demographic trends and predict population growth in different regions.
- **Key Features**:
  - Global population distribution by country and region
  - Projections for population growth over the next decade
  - Country-specific population density and growth rate analysis
- **Files**:
  - `Population_Estimate_Dashboard_2025.pbix` – The Power BI dashboard file
  - `README.md` – Project description
  - `/assets/` – Screenshots and GIFs of the dashboard
